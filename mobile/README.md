# VOLT Mobile - Professional Athlete Ecosystem

##  Overview
VOLT is a premium sports performance platform designed for athletes to discover elite gear, analyze their performance, and dominate the arena.

##  Tech Stack
- **Framework**: Expo / React Native
- **Styling**: StyleSheet (Vanilla CSS approach)
- **Icons**: Lucide React Native
- **Animations**: React Native Reanimated
- **Gradients**: Expo Linear Gradient

##  Project Structure
- `assets/`: Image assets and animations
- `src/constants/`: Static data and configurations
- `src/theme/`: Global theme constants and colors
- `App.js`: Main application entry and root navigation

##  Features
- **Dual Mode UI**: Seamless switching between Beginner (Explore) and Pro Flow.
- **Position Analysis**: Interactive pitch visualization for tactical gear selection.
- **Premium Search**: Real-time gear and athlete discovery.
- **Vibrant UX**: 60fps animations and high-fidelity dark mode aesthetics.
